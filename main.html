Question-1: What are the different ways to select an element in the DOM?

The different ways to select an element is getElementById(), getElementsByClassName(), getElementsByTagName(), and getElementsByName(). More flexible options include querySelector().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gets HTML content, including tags. innerText gets only visible text, ignoring hidden elements, while textContent gets all text, including hidden elements, without parsing HTML.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for its child elements using event bubbling. Instead of adding event listeners to multiple child elements, you attach a single listener to a common ancestor and use event.target to identify the clicked element.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where an event starts from the target element and propagates up to its ancestors in the DOM hierarchy. It allows parent elements to listen for events triggered by their child elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Use document.createElement("tagName") to create an element, and add it using appendChild(), append(), or prepend(). Remove elements with remove() or removeChild().
